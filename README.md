# alfred-emmet

emmet expand command for Alfred worklow.

## Install

```
$ npm install --global alfred-emmet
```

### Usage

1. In Alfred, type 'emmet'. 
2. Input your query (ex. "ul#nav>li*4") 
3. Enter to copy to clipboard
